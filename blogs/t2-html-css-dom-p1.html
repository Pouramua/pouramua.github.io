HTML, CSS and DOM P1

Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?
HTML is like the structure of your house the walls, the roof, the doors etc. CSS is the colour schemes and designs of the things in your house, like the colour of the wallpaper. Dom is the thing that controls the certain stuff in your house like the remote to turn the tv on or open the garage door.
What is meant by boxifying design?
It meants that everything on your screen is within a box, every single thing on your screen is in their own individual boxes, even cicles and triangles.
What is the box model?
HTML elements are considered as boxes. In CSS, the term "box model" is used when talking about design and layout. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.
