What 'scope' is, and how it works in JavaScript.



Heres some of the resources we were given about Scope, last week & a youtube video about the The Basics of Scope in JavaScript. After watching this video I learnt that there are two types of scope, 'Local Scope' & 'Global Scope'.
Local Scope

A local scope is a variable which is either a variable declared within the function or is an argument passed to a function. Global Scope

A global scope means that it is visible (hence accessible) throughout the program, unless shadowed. The set of all global variables is known as the global environment or global state.
