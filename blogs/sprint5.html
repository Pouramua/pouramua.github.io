  • .map()
    just like filter map is a function that will take an array and transform that into an array of the same length it will also go through an array but unlike filter it doesnt throw theobjects away it transforms them.
  • .filter()
    filter is a function on the array that accepts another function as its arguement which it will use to return a new filter but it will also transform an array into a smaller array.

    reduce can be used to express any list transformation you can always fall back on it if you cant find a pre build list transformation that fits your problem.
